Acerca de Venta por Catálogo

Este proyecto es una aplicación web desarrollada con Flask que simula un sistema de venta por catálogo. Los usuarios pueden registrarse, iniciar sesión y gestionar un inventario de productos con imágenes.

Características

  • Registro y autenticación de usuarios.
  • Gestión de productos con imágenes.
  • Búsqueda y visualización de productos.
  • Interfaz intuitiva y responsive.

Tecnologías Utilizadas

  • Python con Flask.
  • MySQL para la base de datos.
  • Bootstrap para el diseño responsive.
  • HTML, CSS y JavaScript.
